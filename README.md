# Machine-Learning-Project
This project is part of a data science and machine learning study focused on applying predictive modeling techniques to real-world health-related data. It explores how structured data can be used to extract meaningful insights and support decision-making in public health contexts.

## Goal of the Project
The goal of this project is to analyze and predict alcohol consumption behavior using demographic, lifestyle, and physiological data.
Through machine learning techniques, the project aims to identify patterns associated with risky alcohol consumption and evaluate how body signals and personal characteristics contribute to predictive performance.

The objective is not only to build an accurate predictive model, but also to ensure interpretability and transparency in order to better understand the factors influencing alcohol consumption.

## Programming Language
The entire project was developed using Python, ensuring reproducibility and integration within a structured machine learning workflow.

## Libraries Used
- NumPy – numerical computation
- Pandas – data manipulation and preprocessing
- Matplotlib / Seaborn – data visualization
- Scikit-learn – machine learning models, preprocessing pipelines, model evaluation
- XGBoost – gradient boosting model implementation
- SHAP – model interpretability and feature importance analysis
